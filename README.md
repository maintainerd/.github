<h1 align="center">Maintainerd GitHub Profile</h1>

This repository powers the public profile for the `maintainerd` GitHub organization.

Maintainerd is an open-source, self-hostable catalog of cloud-style product and platform services. Developers can run each service standalone, or connect services through M9d-Core for a shared control plane.

The organization profile content lives in [profile/README.md](profile/README.md).
